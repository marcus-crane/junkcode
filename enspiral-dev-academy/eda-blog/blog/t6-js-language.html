Parentheses ()

They can be used to group expressions: b = (1 + 2) * 3.

They can be used to contain expressions: if (b > 5) or while (b < 5).

They can also be used to contain parameters for a function: function(parameter) or just function() by itself.

Brackets []

Brackets are used to initialise and access arrays. They're also used in one of the two ways you can access objects.

If you had an array like Array = [1, "Fruit", true], you would access it using brackets.

Array[0] = 1, Array[1] = "Fruit", Array[2] = true

The brackets themselves store the array position that you want to access. Remember that array positions start at 0 instead of 1!

If you have an object you're trying to access, brackets are used in bracket notation.

It's as simple as listing the object name and then the key you want to access in brackets: obj["objectKey"]

It might seem easier to use dot notation obj.objectKey but bracket notation is super handy if you want to use a variable as the object's key name.

Braces {}

No, not like braces for your teeth!

Braces in Javascript are used for defining where a function starts and ends. Simple as that.

if (1 < 2) {
    console.log("2 is bigger than 1");

Single Quotes ' ' and Double Quotes " "

I've put these together because as simple as it seems, this one can get a bit tricky.

For the most part, single and double quotes can be used interchangably

var string = "Words"; is just as valid as var string = 'Words';

You can also be used to do quotes within a string literal as such:

var string = 
"She said 'It's a quote within a quote'"

and inversely

var string =
'She said "It's a quote within a quote"'

Really, it's personal preference but if you tend to use apostrophes a lot in your strings, double quote are your friend.

Be careful though because if you're doing JSON at all, you MUST use double quotes!
