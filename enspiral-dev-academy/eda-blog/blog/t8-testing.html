What is scope?

When you have a value or expression, scope refers to the context in which said values and expressions are visible.

Let's say I have the following piece of code:

function returnDog() {
    var animal = "dog";
    return animal;
}

The animal variable is defined inside the returnDog() function. When run, returnDog() will return the value of animal which in this case is "dog"

Now what if I tried to change the value of animal to say, "cat". You would think returnDog() would turn into some sort of returnCat() instead, right?

function returnDog() {
    var animal = "dog";
    return animal;
}

animal = "cat";

returnDog() // value is still "dog"

But how is that?! We changed the animal variable to "cat" right before we called the function!

That's where scope comes in. The animal variable contained within returnDog() is only visible to the other parts of the code within the function.

What ends up happening is that we've defined two lots of animal variables. One is within the function which contains the value of "dog" and the other sits outside of the function with the value of "cat".

Since functions only exist as they're being called, the animal variable within returnDog() is able to exist seperately from any other potential variables. It's possible to have hundreds of animalvariables within hundreds of functions (although why would you) since they'd never interact with one another.

While you can't alter variables declared within functions from the outside, you can do the reverse.

i = 0;
function count() {
    j = " is a number";
    return i + j;
}

This code would return "0 is a number".

From within count(), I could set i equal to "hotdogs" and we could get "hotdogs is a number" as a result.

We could then set j to equal " are a food" so that our code makes a bit more sense

Due to Javascript scope however, while j would still be equal to " are a food", once we run count(), the output would be the same as we are left with two different j variables: once outside the function and one within it.

The same applies to functions within functions and so on. Scope works from the present level upwards.
